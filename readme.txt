== Description ==

The PEBO Dropdown Spheres free version, is a ligth weigh widget to show selected post categories and a drpdown 
menu that shows links to non empty subcategories.

== Use Instructions ==

*   7 different effects for selecting one to be used.
*   Check on each category wanted to be shown at the front end.
*   Each effect comes with 3 default images to select on each category and show how the effects works.
*   To customize the background image or use color make a donation using the donatino link 
    and get the fully featured pro version.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Find your widget within your wigets area an assign it to a secction of your page.
1. Click on the plugin name to configure your desired options.
